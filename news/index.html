Wetator version 0.9.11 is available. (2013-06-16)

Wetator version 0.9.11 is available.

This release improves the handling of zip'ed and 'unsupported' content. Wetator now automatically unpacks zip'ed content; you can use assert-content transparently. For 'unknown' content Wetator handles the bytes as plain ascii.
We fixed some bugs with the preview in the report. Now it works with Chrome and IE also. The file chooser got some improvements; he remembers the last selection, window position and size.
Of course we use the latest HtmlUnit snapshot which brings an updated Javascript implementation (rhino), an improved css parser and many bug fixes in HtmlUnit itself.
As always please have a look at the release notes.

Enjoy testing...

Posted by RBRi

Wetator version 0.9.10 is available. (2013-03-17)

Wetator version 0.9.10 is available.

This release includes many bug fixes (mainly in HtmlUnit) and some great improvements for Wetator itself. We improved the internal wait implementation for ajax requests, the html report now includes a page snapshot preview and the setting of control values now handles automatic focus changes correctly.
Again we use the latest HtmlUnit snapshot which brings a newer Javascript implementation, an improved css parser (including CSS3 selector support) and many bug fixes.
Starting with this release the ant task is packed into a separate jar file.
Please have a look at the release notes.

Happy testing...

Posted by RBRi
