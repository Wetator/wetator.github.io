Ant Task

Download & Installation

The ant task requires a different Wetator distribution.

  • Download the latest stable version.
  • Create a new directory (or use the directory from the command line installation) and unzip the file into it (preserve subdirectories).
  • The file build.xml in the templates directory contains sample ant targets that runs the wetator with a collection of files. You only have to adjust the paths.
