Glossary

Test Case

test definition as a script of Commands

Test File

a file in a supported format (e.g. wet, wett, xlsx), holding (parts of) a Wetator Test Case
one or multiple Test Files comprise one Test Case

Command

the smallest unit of a Test Case

Test

the execution of a Test Case in a particular browser

Test Step

the execution of a Command during a Test
